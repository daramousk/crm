=============================
membership_contract_hierarchy
=============================

This is a glue module between membership_contract and
partner_multi_relation_hierarchy. When installed a partner will be a member
if any partner above in the hierarchy is a member.

Usage
=====

To use this module, you must make sure that the Odoo membership is
uninstalled.

Module should be automatically installed when its requisites are installed.
